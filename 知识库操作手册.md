## 二、日常核心操作

### 2.1 查询知识库（最常用）

**入口**：侧边栏 → 在线知识库

**使用场景**：业主来电咨询时，快速查找标准答案。

**操作步骤**：

1. 在搜索框输入关键词（如"物业费""漏水""停车"）
2. 点击分类标签筛选（如"物业缴费""报修服务"）
3. 点击知识卡片查看详情
4. 点击"复制答案"直接粘贴给业主

**技巧**：
- 搜索支持模糊匹配，输入问题的一部分即可
- 可同时使用关键词搜索 + 分类筛选缩小范围

---

### 2.2 智能问答（AI 辅助）

**入口**：侧边栏 → 智能问答

**使用场景**：知识库搜不到时，用自然语言提问获取 AI 回答。

**操作步骤**：

1. 在输入框输入问题（如"装修施工时间有什么规定？"）
2. 点击"发送"或按 Enter
3. 等待系统返回回答
4. 查看回答下方的命中状态和相似度

**回答状态说明**：

| 状态 | 含义 | 建议 |
|------|------|------|
| 已命中 | 找到相似知识，回答可靠 | 可直接使用 |
| 未命中 | 知识库无匹配，回答可能不准 | 建议人工核实后回复 |

**反馈操作**：
- 回答准确 → 点击"有帮助"
- 回答不准 → 点击"没帮助"（会进入反馈处理流程）
- 问题复杂 → 点击"需要人工处理"

---

### 2.3 查看咨询记录

**入口**：侧边栏 → 我的咨询记录

**使用场景**：回溯之前的问答记录。

**可查看内容**：
- 咨询时间
- 用户问题和系统回答
- 命中分类和相似度
- 命中状态（已命中/未命中）

---

## 三、管理端操作（需管理员账号）

### 3.1 处理反馈

**入口**：管理端侧边栏 → 反馈处理

**使用场景**：处理用户对回答质量的反馈。

**操作步骤**：

1. 查看反馈列表，关注"待处理"状态的反馈
2. 点击"查看详情"了解用户问题和 AI 回答
3. 根据情况选择处理方式：

| 操作 | 适用场景 |
|------|----------|
| 标记已处理 | 已电话回访用户、已核实回答无误 |
| 加入知识库 | 发现知识库缺失该问题的标准答案 |
| 忽略 | 重复反馈、用户误操作 |

**将反馈加入知识库**：

1. 点击"加入知识库"
2. 修改标准问题（使其更规范）
3. 填写准确的标准答案
4. 选择分类、添加关键词
5. 保存后到"知识库维护"重建向量库

---

### 3.2 处理未命中问题

**入口**：管理端侧边栏 → 咨询日志

**使用场景**：发现智能问答频繁未命中，补充知识库。

**操作步骤**：

1. 点击"仅查看未命中问题"
2. 找到高频或重要的未命中问题
3. 点击"加入知识库"
4. 填写标准问题和答案
5. 保存后重建向量库

---

### 3.3 维护知识库

**入口**：管理端侧边栏 → 知识库维护

**新增问答**：

1. 点击"新增问答"
2. 填写标准问题、标准答案
3. 选择分类、添加关键词
4. 状态设为"已发布"
5. 保存后点击"重建向量库"

**批量导入**：

1. 点击"批量导入"
2. 下载 CSV 模板
3. 按模板填写数据
4. 上传并导入
5. 重建向量库

> **重要**：新增或修改知识后，必须"重建向量库"才能生效！

---

### 3.4 重建向量库

**入口**：管理端侧边栏 → 向量库维护

**何时需要重建**：
- 新增/编辑/删除问答后
- 处理反馈并加入知识库后
- 批量导入问答后

**操作**：点击"重建向量库"，等待完成即可。

## 四、常见问题速查

| 问题 | 解决方案 |
|------|----------|
| 搜索不到相关内容 | 换关键词试试，或用智能问答提问 |
| AI 回答不准确 | 在知识库补充标准答案，重建向量库 |
| 新增知识后没生效 | 确认状态为"已发布"，并重建向量库 |
| 登录后提示修改密码 | 按要求修改，需包含字母和数字 |
| 页面提示未登录 | 重新登录，或清除浏览器缓存 |

## 五、日常工作流程建议

### 每日工作

1. 登录管理端，查看"反馈处理"是否有待处理反馈
2. 进入"咨询日志"，筛选未命中问题
3. 将高频问题补充到知识库
4. 重建向量库

### 接到业主咨询时

1. 先在"在线知识库"搜索
2. 搜不到则用"智能问答"提问
3. 回答命中且准确 → 直接回复业主
4. 回答未命中或不准 → 人工核实后回复，并反馈"没帮助"

### 发现知识库缺失时

1. 在"知识库维护"新增标准问答
2. 或在"反馈处理"/"咨询日志"中将问题加入知识库
3. 重建向量库
4. 用智能问答验证新知识是否生效
